# Scene

## Constructor

```js
new Scene(sceneName: String);
```

## Properties

> #### .svg: d3 Selector

d3 selector of this scene's svg tag.

> #### .name: String

Name of the scene.

## Methods

> #### .destroy(delay)

##### Parameters

> > delay: Number

Time delay (milliseconds).

Remove this scene's svg tag from HTML structure (after fisnishing fading out).
